## Билет 1 — Мобильные приложения (Android)

**Задание (для поиска):** «...разработке мобильных приложений... интегрированная среда разработки, предназначенная для создания приложений для любых устройств на Android... изменённую версию свободного набора компиляторов... эмулятор 3D графики... программа-эмулятор, которая создает среду различных операционных систем...»

**Хорошо — установка ПО:**
```bash
# Android Studio (IDE для Android)
sudo snap install android-studio --classic

# Кросс-компилятор (изменённая сборка GCC)
sudo apt update && sudo apt install -y mingw-w64 build-essential

# Ускорение эмулятора / 3D-графика
sudo apt install -y qemu-kvm libvirt-daemon-system mesa-utils

# Эмулятор ОС (программа создающая среду других ОС)
sudo apt install -y virtualbox

# Защита от хакерских атак
sudo apt install -y clamav clamav-daemon ufw
sudo ufw enable
```

**Отлично — руководство пользователю (Android Studio):**
- **Наименование:** Android Studio — официальная IDE для разработки Android-приложений.
- **Краткое описание:** объединяет редактор кода, эмулятор устройств, отладчик и сборщик Gradle.
- **Запуск:** через меню приложений либо командой `studio.sh` из каталога установки/снапа.
- **Выход:** File → Exit или закрытие окна (штатное сохранение проекта).
- **Функции:** создание/импорт проекта, визуальный редактор макетов (Layout Editor), эмулятор AVD, отладчик, профилировщик, Gradle-сборка, Git-интеграция.
- **Приёмы работы:** создать новый проект → выбрать шаблон Activity → настроить AVD (виртуальное устройство) → Run ▶ для запуска в эмуляторе → Logcat для просмотра логов.

## Билет 4 — 3D игры (C#, игровой движок)

**Задание (для поиска):** «...3D игр для ПК и консолей... среда разработки, встроенная в игровой движок и поддерживающая язык C#... графический API для рендеринга 3D графики... программа для создания и редактирования 3D моделей... инструмент для управления проектами и версиями игровых ассетов...»

**Хорошо — установка ПО:**
```bash
# Unity Hub (движок + IDE с C#)
wget -qO- https://hub.unity3d.com/linux/keys/public | gpg --dearmor | sudo tee /usr/share/keyrings/Unity_Technologies_ApS.gpg > /dev/null
sudo sh -c 'echo "deb [signed-by=/usr/share/keyrings/Unity_Technologies_ApS.gpg] https://hub.unity3d.com/linux/repos/deb stable main" > /etc/apt/sources.list.d/unityhub.list'
sudo apt update && sudo apt install -y unityhub

# Графический API (Vulkan)
sudo apt install -y vulkan-tools libvulkan1

# 3D-моделирование
sudo apt install -y blender

# Управление версиями ассетов
sudo apt install -y git git-lfs
git lfs install

# Защита от эксплойтов/сетевых атак
sudo apt install -y ufw fail2ban
sudo ufw enable
```

**Отлично — руководство пользователю (Git LFS):**
- **Наименование:** Git LFS (Large File Storage) — расширение Git для версионирования крупных бинарных ассетов.
- **Краткое описание:** хранит большие файлы (модели, текстуры) отдельно, в репозитории — только указатели.
- **Запуск:** работает поверх обычных команд `git` после `git lfs install`.
- **Выход:** не требуется отдельного завершения — интегрирован в Git.
- **Функции:** отслеживание типов файлов (`git lfs track "*.png"`), пуш/пул больших файлов, экономия места в репозитории.
- **Приёмы работы:** `git lfs install` → `git lfs track "*.fbx"` → `git add .gitattributes` → обычный `git add/commit/push`.

## Билет 5 — DevOps (CI/CD инфраструктура)

**Задание (для поиска):** «...команда DevOps... инфраструктура для сборки и развертывания... продвинутый текстовый редактор... языков сценариев (Bash, Python), системных языков (Go, Rust)... платформа для автоматизации сборки, тестирования и развертывания (CI/CD)... инструмент для контейнеризации... система мониторинга состояния серверов...»

**Хорошо — установка ПО:**
```bash
# Редактор кода
sudo snap install code --classic

# CI/CD платформа (Jenkins)
sudo apt update
sudo apt install -y openjdk-17-jre
curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key | sudo tee \
  /usr/share/keyrings/jenkins-keyring.asc > /dev/null
echo "deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc]" \
  "https://pkg.jenkins.io/debian-stable binary/" | sudo tee \
  /etc/apt/sources.list.d/jenkins.list > /dev/null
sudo apt update && sudo apt install -y jenkins

# Контейнеризация
sudo apt install -y docker.io
sudo systemctl enable --now docker

# Мониторинг
sudo snap install prometheus
sudo snap install grafana

# Централизованная безопасность/аудит
sudo apt install -y fail2ban ufw
```

**Отлично — руководство пользователю (Jenkins):**
- **Наименование:** Jenkins — платформа автоматизации CI/CD.
- **Краткое описание:** автоматизирует сборку, тестирование и развёртывание приложений через настраиваемые пайплайны.
- **Запуск:** сервис стартует автоматически (`systemctl status jenkins`), веб-интерфейс — `http://localhost:8080`.
- **Выход:** `sudo systemctl stop jenkins` останавливает сервис.
- **Функции:** создание Job/Pipeline, интеграция с Git, плагины для сборки/тестов/деплоя, распределённые агенты, отчёты о сборках.
- **Приёмы работы:** New Item → Pipeline → указать репозиторий/Jenkinsfile → Build Now → просмотр Console Output.
